# 5
Объектно-ориентированное программирование
+ вариант 30
- a) Класс - однонаправленный список list. Дополнительно перегрузить
следующие операции:
1. (+) - добавить элемент в конец (list+item);
2. -- - удалить элемент из конца (типа list--);
3. != - проверка на неравенство.
- b) Разработайте иерархию классов микроорганизмы (приватные члены
сконструировать самостоятельно). Добавьте статические члены по подсчету
минимального размера микроорганизма. Разработайте не менее 2-х
виртуальных функций.
- c) Создайте шаблонный класс однонаправленного списка
микроорганизмов и символов. Продемонстрируйте работу класса и все его
операции.
